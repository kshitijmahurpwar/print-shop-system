# Print Shop Management System

A local network-based print management system for print shops.

## Customer Flow

1. Scan QR code
2. Open local web app
3. Upload file
4. Select print options
5. Receive token

## Vendor Flow

1. Open dashboard
2. View pending jobs
3. Preview files
4. Update status
5. Print documents

## Tech Stack

Frontend:

* Angular
* TailwindCSS

Backend:

* Django
* Django REST Framework

Database:

* SQLite initially
* MySQL later

## MVP Features

* File upload
* Token generation
* Vendor dashboard
* Status tracking

## Future Features

* Payments
* Notifications
* Multi-vendor support
* Cloud sync
